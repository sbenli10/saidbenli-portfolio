🌐 Responsive Kişisel Portföy Web Sitesi
Bu proje, HTML, CSS ve JavaScript kullanılarak oluşturulmuş tamamen responsive (mobil uyumlu) bir kişisel portföy web sitesidir. Kullanıcı dostu ve modern bir arayüz ile tasarlanmıştır.

✨ İçerik
Anasayfa (Home)

Hakkımda (About)

Beceriler (Skills)

Nitelikler / Eğitim (Qualification)

Hizmetler (Services)

Portföy (Portfolio)

Proje Önerisi Alanı (Project in mind)

Geri Bildirimler (Testimonial)

İletişim (Contact)

Alt Bilgi (Footer)

📌 Özellikler
Modern ve şık tasarım

Duyarlı (responsive) yapı

Hafif ve performans odaklı

JavaScript ile tema değiştirme (light/dark mode)

Swiper.js ile kaydırılabilir bileşenler

Formspree.io ile iletişim formu entegrasyonu

Kolayca özelleştirilebilir yapı
